半透明边框

给一个容器同时设置边框和背景时，可能会发现背景会延伸到边框所在的区域下层

可以通过backgroud-clip来调整

多重边框

box-shadow 方案

box-shadowj基本用法：

  • 基本语法

    • inset

      不使用inset，默认阴影在边框外，即阴影向外扩散

      使用inset后，阴影在边框内（即使是透明边框），即阴影向内扩散，背景之上内容之下

    • offset-x offset-y

      offset-x设置水平偏移量，正值阴影则位于元素右边，负值阴影则位于元素左边

      offset-y设置垂直偏移量，正值阴影则位于元素下方，负值阴影则位于元素上方

      如果两者都是0，那么阴影位于元素后面。这时如果设置了blur-radius或spread-radius则有模糊效果，需要考虑inset

    • blur-radius

      该值越大，模糊面积越大，阴影就越大越淡，不能为负值。默认为0，此时阴影边缘锐利。需要考虑inset

    • spread-radius

      取正值时，阴影扩大；取负值时，阴影收缩。默认为0，此时阴影与元素同样大。需要考虑inset

    • color

      如果没有指定，则由浏览器决定——通常是color的值，不过目前Safari取透明

  • box-shadow支持逗号分隔，可以创建任意数量的阴影，扩张半径以最里层为基准

!需注意，此方法生成的假边框不会影响鼠标事件

outline 方案

如果只需要两层边框，可以先设置常规边框，再用outline属性产生外层的边框

使用outline可以通过outline-offset属性控制它和元素的边距，可以为负

使用outline生成的边框不一定会贴合border-radius的圆角

边框内圆角

通常可以使用两个嵌套的div实现

在只使用一个div时，可以使用outline，但是使用outline生成的边框不一定会贴合border-radius的圆角，所以可以用box-shadow填充描边和圆角之间的空隙
