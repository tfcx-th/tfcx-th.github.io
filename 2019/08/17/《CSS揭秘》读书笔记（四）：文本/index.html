连字符断行

在对文本进行两端对齐时，可能会造成每行单词不一致的结果，可以通过设置hyphens属性来为文本换行时设置连字符

hyphens总共有三种取值：none、manual和auto，其中manual为初始值，表示需要手工插入软连字符&shy;

定义列表换行

在定义列表中，我们往往会想要名和值在同一行，然而由于<dt>和<dd>都是块级元素，结果会导致名和值各占一行

可以设置紧跟着每个<dd>之后的<dt>的::before伪元素，并且保留空白符和换行

如果有相邻的<dd>，则需要添加,，并利用负外边距减少逗号前的空格
